# SQL Exercises

A set of SQL scripts covering database design, query writing, joins, stored procedures, and real-world schema modelling.

## Files

| File | Description |
|------|-------------|
| `FITNESS.sql` | Fitness centre database schema and queries |
| `GARMENTS.sql` | Garments/clothing store schema and sample queries |
| `joins.sql` | JOIN exercises — INNER, LEFT, RIGHT, FULL OUTER |
| `48.sql` | Exercise 48 — advanced query problems |
| `Q 43 and 44 of PF.sql` | PF (Practice File) questions 43 & 44 |
| `Q42 of PF.sql` | PF question 42 |
| `Q45 of PF.sql` | PF question 45 |

## Getting Started

```bash
git clone https://github.com/DrOneEyed/sql-exercises.git
cd sql-exercises
# Run any script in MySQL:
mysql -u root -p < FITNESS.sql
```

## Topics Covered

- DDL — `CREATE TABLE`, `ALTER`, `DROP`
- DML — `INSERT`, `UPDATE`, `DELETE`
- DQL — `SELECT` with filtering, grouping, ordering
- Joins — INNER, LEFT, RIGHT, CROSS
- Aggregate functions — `COUNT`, `SUM`, `AVG`, `MAX`, `MIN`
- Subqueries and nested queries
- Stored procedures and functions
